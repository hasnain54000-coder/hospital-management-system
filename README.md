🏥 Hospital Patient Management System (HPMS)
A comprehensive, dark-themed management solution designed to streamline hospital workflows, patient data tracking, and departmental efficiency.

🚀 Core Features
Integrated Dashboard: Real-time statistics for total patients, OPD registrations, Emergency cases, and Pharmacy records.

Departmental Management: Specialized sections to handle OPD, Emergency, and Pharmacy patient flows.

Dynamic Data Handling: Add, view, and manage patient records with instant dashboard updates.

Advanced UI/UX: A modern, responsive sidebar-based navigation with a sleek "Advanced Dark Theme" for professional environments.

Search & Filter: Quickly locate patient records within specific departments.

🛠️ Tech Stack
Frontend: HTML5 & CSS3 using an advanced dark-mode color palette (Deep Navy and Slate).

Iconography: Font-Awesome integration for professional medical and navigation icons.

Logic: Vanilla JavaScript for state management, dynamic DOM manipulation, and real-time dashboard calculations.
